KindEditor

Configuration

width

Set the width of the editor.

  • Data type: String
  • Default value: the width of the textarea

Example:

K.create('#id', {
    width : '700px'
});

height

Set the height of the editor.

  • Data type: String
  • Default value: the height of the textarea

Example:

K.create('#id', {
    height : '300px'
});

minWidth

Set the minimal width of the editor.

  • Data type: Int
  • Default value: 650

minHeight

Set the minimal height of the editor.

  • Data type: Int
  • Default value: 100

items

Set the toolbar of editor.

  • Data type: Array
  • Default value:

.. index:: themeType

.. _themeType:

themeType

指定主题风格，可设置"default"、"simple"，指定simple时需要引入simple.css。

  • 数据类型: String
  • 默认值: "default"

示例:

.. sourcecode:: html

<link rel="stylesheet" href="../themes/default/default.css" />
<link rel="stylesheet" href="../themes/simple/simple.css" />
<script charset="utf-8" src="../kindeditor.js"></script>
<script charset="utf-8" src="../lang/zh_CN.js"></script>
<script>
    var editor;
    KindEditor.ready(function(K) {
        editor = K.create('#editor_id', {
            themeType : 'simple'
        });
    });
</script>
